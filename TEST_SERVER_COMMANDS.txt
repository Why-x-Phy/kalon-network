=== KALON NETWORK TEST SERVER COMMANDS ===

Schritt 1: Repository klonen
git clone https://github.com/Why-x-Phy/kalon-network.git
cd kalon-network

Schritt 2: Binaries bauen
go build -o build-v2/kalon-node-v2 cmd/kalon-node-v2/main.go
go build -o build-v2/kalon-miner-v2 cmd/kalon-miner-v2/main.go
go build -o build-v2/kalon-wallet cmd/kalon-wallet/main.go

Schritt 3: Test-Setup vorbereiten
pkill -9 -f kalon || true
rm -rf data-v2/testnet wallet.json
mkdir -p data-v2/testnet

Schritt 4: Wallet erstellen
echo "" | ./build-v2/kalon-wallet create 2>&1 | grep "Address:"
WALLET=$(cat wallet.json | jq -r .address)
echo "Wallet: $WALLET"

Schritt 5: Node starten
./build-v2/kalon-node-v2 -datadir data-v2/testnet -genesis genesis/testnet.json -rpc :16316 > /tmp/kalon_node.log 2>&1 &
sleep 7

Schritt 6: Minen starten
timeout 10 ./build-v2/kalon-miner-v2 -wallet "$WALLET" -threads 1 -rpc http://localhost:16316

Schritt 7: Balance prÃ¼fen
sleep 2
BALANCE=$(curl -s http://localhost:16316/rpc -H "Content-Type: application/json" -d "{\"jsonrpc\":\"2.0\",\"method\":\"getBalance\",\"params\":{\"address\":\"$WALLET\"},\"id\":2}" | jq -r .result)
echo "BALANCE: $BALANCE"

Schritt 8: Node stoppen
pkill -9 -f kalon

=== Erwartetes Ergebnis ===
Balance sollte > 0 sein (z.B. 5000000 oder mehr)
